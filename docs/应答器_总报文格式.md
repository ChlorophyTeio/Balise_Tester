## 1. 帧结构与变量定义

| 序号 | 变量名 | 位数 (Bits) | 变量作用与定义 |
| :--- | :--- | :--- | :--- |
| **1** | **Q_UPDOWN** | 1 | 信息传送方向：0=车对地，1=地对车 |
| | **M_VERSION** | 7 | 语言/代码版本编号，示例 0010000=V1.0 |
| | **Q_MEDIA** | 1 | 传输媒介：0=应答器，1=环线 |
| | **N_PIG** | 3 | 本应答器在组内的位置：000=第1个 ... 111=第8个 |
| | **N_TOTAL** | 3 | 应答器组数量：000=1个 ... 111=8个 |
| | **M_DUP** | 2 | 与前后应答器信息关系：00=不同/01=与后相同/10=与前相同 |
| | **M_MCOUNT** | 8 | 报文计数器：无源=255，有源默认=252，LEU=0，TCC默认=255，TCC故障=253，禁用值=254 |
| | **NID_C** | 10 | 地区编号：高7位=大区，低3位=分区 |
| | **NID_BG** | 14 | 应答器标识号：高6位=车站编号，低8位=应答器编号 |
| | **Q_LINK** | 1 | 链接关系：0=不被链接，1=被链接 |
| **2** | **用户信息包** | 772 | 核心数据区，包含 ETCS/CTCS 信息包 |
| **3** | **结束标志** | 8 | 固定二进制 11111111 |

### 1.1 报文计数器 (M_MCOUNT) 规则
- 无源应答器：255；有源默认：252；LEU 默认：0；TCC 默认：255；TCC 故障：253；禁用：254。
- 一致性校验：应答器组内除 255 以外计数一致，报文才可用。

## 3. 数据包有效范围

### 3.1 区间闭塞分区应答器组
- 链接信息 ETCS-5：正向链接前方相邻两个组；反向链接相邻一个组。
- 轨道区段 CTCS-1：本组起至前方第二个区间组【Q】再延长一个制动距离；坡度/速度 ETCS-21/27 同范围。
- 特殊区段 ETCS-68：按分相区发送原则。
- 文本信息 ETCS-72：进站前3个闭塞分区显示，出站进入区间后消失。
- 位置信息 ETCS-79：提供本应答器公里标。

### 3.2 进站应答器组
- 临时限速 CTCS-2：按规范。
- 链接 ETCS-5：正线接车至区间第一架通过信号机范围；侧向接车至出站信号机范围；反向发车链接反方向第一个组。
- 轨道区段 CTCS-1/坡度 ETCS-21/速度 ETCS-27：依场景覆盖至区间或制动距离范围。
- 特殊区段 ETCS-68：描述距最近分相区。
- 里程 ETCS-79：本组里程。

### 3.3 出站应答器组
- 轨道区段 CTCS-1：至下行方向第一个区间组【Q】再延长一个制动距离；反向预告同范围。
- 链接 ETCS-5：出站范围内所有组；反向预告链接前方出站信号机组。
- 坡度/速度 ETCS-21/27：与发车轨道区段一致。
- 特殊区段 ETCS-68：描述距最近分相区。
- 里程 ETCS-79：本组里程。

## 4. 信息包字段汇总

### 4.1 报文总架构
- 包头信息帧：帧标志、位置、编号等控制字段（固定结构）。
- 用户信息包：按需组合 20 类功能包（ETCS 15 类 + CTCS 5 类）。
- 信息结束标志：固定 11111111。

### 4.2 包头字段
| 分类 | 字段名称 | 变量名 | 位数 | 核心说明 | 默认值/规则 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 帧标志 | 信息传输方向 | Q_UPDOWN | 1 | 0=车对地，1=地对车 | 应答器默认1 |
| 帧标志 | 代码版本 | M_VERSION | 7 | 语言/代码版本编号 | 0010000=V1.0 |
| 帧标志 | 传输媒介 | Q_MEDIA | 1 | 0=应答器，1=环线 | 默认0 |
| 位置与数量 | 组内位置 | N_PIG | 3 | 组内序号 | 000=第1个、111=第8个 |
| 位置与数量 | 组内总数量 | N_TOTAL | 3 | 组内应答器总数 | 000=1个、111=8个 |
| 数据关系 | 信息重复标识 | M_DUP | 2 | 与前后应答器信息关系 | 00/01/10 |
| 计数器 | 报文计数器 | M_MCOUNT | 8 | 报文计数标识 | 无源=255/有源=252/LEU=0/TCC正常=255/TCC故障=253 |
| 标识编号 | 地区编号 | NID_C | 10 | 高7位=大区，低3位=分区 | - |
| 标识编号 | 应答器标识号 | NID_BG | 14 | 高6位=车站编号，低8位=应答器编号 | - |
| 链接关系 | 链接状态 | Q_LINK | 1 | 0=不被链接，1=被链接 | 调车组可设0 |
